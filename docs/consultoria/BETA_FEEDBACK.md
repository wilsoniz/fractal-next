# Beta Feedback

## Regra de priorização

Toda solicitação deve registrar:

- usuário;
- contexto;
- problema observado;
- frequência;
- impacto;
- risco;
- solução sugerida;
- prioridade;
- status;
- evidências;
- data.

## Escala de prioridade

- P0: bloqueia uso, risco de perda de dados ou segurança;
- P1: atrito frequente com alto impacto;
- P2: melhoria relevante de fluxo;
- P3: conveniência;
- P4: preferência estética ou baixa frequência.
